================================================================================
変換のコンセプト
================================================================================

1. 清音 -> 濁音: カッ -> ガッ
   軽やかな音 -> 重厚で力強い音

2. 軽やか -> 重厚: チリン/カラン -> ゴロゴロ
   高音で明るい音 -> 低音で重い音

3. 単発 -> 連続: カッ -> ドンドン/ガンガン
   短い音 -> 連続した激しい音

4. 明るい -> 暗い: キラキラ -> ゴロゴロ/ガンガン
   高音強調 -> 低音強調
