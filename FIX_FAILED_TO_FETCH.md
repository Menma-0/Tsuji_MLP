# "Failed to fetch" エラーの修正方法

## 🔧 クイック修正（3ステップ）

### ステップ1: バックエンドAPIサーバーを起動

```bash
python api_server.py
```

✅ 成功すると以下のように表示されます:
```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
```

### ステップ2: APIが動作しているか確認

ブラウザで http://localhost:8000 を開く

✅ 以下のように表示されればOK:
```json
{"status":"ok","message":"Onoma2DSP API Server is running","version":"1.0.0"}
```

### ステップ3: フロントエンドを起動

**別のターミナル**で:
```bash
cd frontend
npm run dev
```

✅ http://localhost:3000 が起動します

---

## 🚀 もっと簡単な方法

**Windowsの場合:**

プロジェクトルートで `start_servers.bat` をダブルクリック

これで両方のサーバーが自動的に起動します！

---

## ✅ 動作確認

1. http://localhost:3000 を開く
2. 音声ファイルをアップロード
3. オノマトペを入力（例: チリン → ゴロゴロ）
4. 「音声を変換」をクリック

### 成功すると:
- ブラウザコンソールに `[Frontend] Response status: 200` と表示
- 処理結果が表示される

### エラーが出る場合:
- ブラウザのコンソール（F12）を開いて確認
- `TROUBLESHOOTING.md` を参照
